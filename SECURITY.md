# Security Policy

## Supported Versions

Newest updates being version 2.0+

## Reporting a Vulnerability

To report a possible or confirmed security vulnerability within the Java Lava Discord bot, please open up a Security Vuln ticket in the tickets section of the Java Lava Support server, which will then execute the Developer Emergency Ping in our tickets via
Java Lava Helper#1735 and the bot will also dm our Lead developer PhillsPhanbh3, saying ⚠️EMERGENCY⚠️ A SECURITY VULN TICKET HAS BEEN OPENED. CHECK NOW!

Please have a lot of detail in your security vulnerability report, with screenshots, what should happen, what is happening, and possible fixes.
